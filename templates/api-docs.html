PROXY Ollama Proxy

The Ollama Proxy runs on a separate port (default 11434) and forwards authenticated requests to the internal Ollama server. All requests require a Bearer token or X-API-Key header containing a valid user License ID.

Configuration
Env Variable Default Description
OLLAMA_PROXY_HOST 0.0.0.0 Bind address
OLLAMA_PROXY_PORT 11434 Proxy listen port
OLLAMA_INTERNAL_URL http://localhost:11435 Internal Ollama server URL
Authentication

Include your License ID in one of these headers:

Authorization: Bearer YOUR-LICENSE-ID
X-API-Key: YOUR-LICENSE-ID
Error Responses
Status Meaning
401 No API key provided
403 Invalid key, insufficient credits, or model not authorized
502 Upstream Ollama server unreachable
GET /api/tags PROXY

List available Ollama models, filtered by the authenticated user's model access list. Served on the proxy port.

  "models": [
